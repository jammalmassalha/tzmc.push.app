מדריך התקנת האפליקציה

אפשר להתקין את האפליקציה ישירות למסך הבית ולעבוד כמו אפליקציה רגילה.

@if (selectedPlatform() === 'android') {
  1. פתח/י את האתר בדפדפן Chrome.
  2. לחץ/י על כפתור התפריט (⋮).
  3. בחר/י "Install app" או "הוספה למסך הבית".
  4. אשר/י את ההתקנה ולחץ/י Add / Install.
} @else {
  1. פתח/י את האתר בדפדפן Safari.
  2. לחץ/י על כפתור השיתוף (Share).
  3. גלול/י ובחר/י "Add to Home Screen".
  4. לחץ/י Add והאפליקציה תופיע במסך הבית.
}

טיפ: אם מופיע כאן כפתור "התקן אפליקציה", אפשר להשתמש בו להתקנה מהירה.
